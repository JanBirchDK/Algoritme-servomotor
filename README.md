# Algoritme-servomotor
I forløbet med at udvikle en overkørsel til modeljernbane, er der udtrykt ønske om at vejbom bevæger sig jævnt. Den servomotor der trækker vejbom, drejer akslen i diskrete trin og ikke kontinuert. Dette repository fremlægger udvikling af en software algoritme, der kan give den mest jævne bevægelse.
<p>I foråret 2022 skrev jeg nogle overvejelser om en ny algoritme. Her i efteråret er der udviklet en algoritme, som håndterer bevægelse af servomotorens arm med reelle tal.</p>
<p>I repository er lagt:</p>
<ul>
<li>Breadboard til udvikling og afprøvning af algoritme.</li>
<li>Målerapport.</li>
<li>Arduino kildekode til udvikling af algoritme.</li>
<li>Arduino kildekode til måling af pulsbredde versus servomotor.</li>
</ul>
